# SWE to AI

A masterclass for software engineers transitioning into ML systems engineering.

**Approach:** Learn by doing and teaching. Every topic becomes a documented lesson with explanations, from-scratch code, mistakes, and exercises.

**Audience:** Working software engineers who use AI tools but want to understand the internals. Not beginners, not PhDs.

## Lesson Structure

Every lesson follows the same format:
1. **Why This Matters** — connection to real ML engineering work
2. **The Intuition** — plain English, analogies from software engineering
3. **The Math** — only what's needed, every symbol explained
4. **Let's Build It** — from-scratch implementation, heavily commented
5. **The Mistakes I Made** — real errors and what they taught
6. **Try It Yourself** — 3–5 exercises, escalating difficulty
7. **The Takeaway** — one paragraph core insight
8. **Resources** — links to materials that helped most
